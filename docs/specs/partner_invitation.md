## パートナー招待機能

### 概要
アプリを利用する2人（パートナー）を紐付けるための機能。
関係は1対1で、1人のユーザーが持てるパートナーは1人だけ。

### 招待の流れ
1. User A がアプリ内で「パートナーを招待」を押すと、招待コード（6〜8文字の英数字）が発行される
2. User A が招待コードを任意の手段（LINE、口頭など）で User B に伝える
3. User B がアプリ内で招待コードを入力すると、2人のパートナー関係が成立する

### ルール
- 招待コードには有効期限がある（24時間程度）
- 招待コードは1回使い切り
- すでにパートナーがいるユーザーは招待・コード入力ができない
- パートナー解除機能は現時点では設けない
- 招待を送信した側（招待者）はパートナー関係において管理モードの変更権限を持つ。パートナー関係テーブルに招待者を記録する
