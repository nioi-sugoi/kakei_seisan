## 画面構成

### 認証画面
- メールアドレス入力画面: メールアドレスを入力し「認証コードを送信」を押下
- OTP認証コード入力画面: 6桁の認証コードを入力して認証完了（300秒の有効期限、再送信・やり直し可能）
- ソーシャルログイン（Google / Apple）: メールアドレス入力画面にボタンを配置

### メイン画面

ナビゲーションはボトムタブ。モードによってタブの表示が変わる。各ユーザーが異なるモードで運用可能。

| モード | タイムライン | パートナー | 設定 |
|--------|:-:|:-:|:-:|
| ソロ | ○ | 非表示 | ○ |
| 共有 | ○ | ○（閲覧のみ） | ○ |
| 管理 | ○（ステータス表示） | ○（承認操作あり） | ○ |

### タイムラインタブ
- 上部に残高サマリーを表示（立替合計 − 預り合計 − 精算済み合計）。プラスなら「家計から受け取り」、マイナスなら「家計へ入金」
- 残高サマリー内に「精算する」ボタン
- 記録と精算を同一タイムラインで時系列表示。月ごとのセクション区切り
- 無限スクロール。初回は最新50件を読み込み、スクロールで追加読み込み（50件ずつ）
- FAB（＋ボタン）から記録登録フォームへ遷移
- 記録・精算タップで詳細画面へ遷移。詳細画面から修正・取り消し・復元が可能

#### カードレイアウト
各記録は以下のレイアウトで表示する:
- 1行目: 種別テキスト（色付き: 立替=青, 預り=橙, 精算=緑）+ 日付（M月D日形式）
- 2行目: ラベル名
- 3行目（任意）: メモ（muted色の小テキスト）
- 右側: 金額（金額は常に正の値で表示。種別の色分けで区別）
- 管理モード時のみ金額の下に承認ステータスインジケーター（✓ 承認済み / ● 承認待ち / ✕ 差し戻し）

#### 修正・取消の表示
- タイムラインには**最新版のみ**表示する。修正された元記録・取消された元記録はタイムラインに表示しない
- **修正された記録**: 日付の右横に鉛筆アイコン（✎）を表示し、修正があったことを示す。タップで詳細画面に遷移し、操作履歴で変更内容を確認できる
- **取消された記録**: カード全体にopacity: 0.5を適用し、金額に取り消し線を付与する。「取消」等のラベルは不要。管理モードでは取消記録にも承認ステータスを表示する

#### フィルター
- **ソロ・共有モード**: 種別フィルターピル（すべて / 立替 / 預り / 精算）
- **管理モード**: フィルターアイコン（横線パターン）+ ステータスピル（すべて / 承認済み / 承認待ち / 差し戻し）をインライン表示。フィルターアイコンをタップするとフィルターモーダルが開き、ステータス・種別・並び替えの詳細条件を指定可能

#### 並び替え（ソート）
- 発生日（新しい順 / 古い順）: occurred_on 基準。デフォルト
- 登録日（新しい順 / 古い順）: created_at 基準

#### 差し戻し
- 差し戻された記録にはパートナーのコメントをカード内にインライン表示する。左ボーダー等の特別な装飾は行わない

#### 詳細画面
- 記録詳細画面: 記録情報カード（種別バッジ・金額・日付・ラベル・メモ）、レシート画像、操作履歴（全バージョンの変更内容を時系列表示）、アクションボタン（修正する / 取り消す / 復元する）
- 精算詳細画面: 記録詳細と同様の構造（種別は「精算」固定）

### 記録登録フォーム
- 種別: 立替 / 預り（セグメント切り替え）
- 金額（日本円）
- 日付（デフォルト: 当日）
- ラベル（自由入力、必須）
- メモ（任意）
- レシート画像（撮影 / ギャラリーから選択、任意、最大2枚）

### 記録修正フォーム
- 金額・ラベル・メモのみ編集可能（種別・日付は変更不可）
- 変更がない場合は送信ボタン無効

### 精算画面（「精算する」ボタンから遷移）
- 現在の残高を表示
- 精算額を入力（デフォルト: 残高全額、変更可能で部分精算に対応。精算額 > 残高は不可）
- 残高がマイナス（預り超過）の場合は「家計に入金する額」として表現する
- 証跡画像の添付が可能（撮影 / ギャラリーから選択、任意、最大2枚）

### パートナータブ
- パートナー未連携時: 招待送信フォーム・送信済み招待一覧・受信した招待一覧を表示（設定タブのパートナー管理と同等のUI）
- パートナー連携済み:
  - パートナーの残高サマリーを表示
  - パートナーの記録・精算をタイムラインで閲覧。カードレイアウト・修正/取消の表示ルールはタイムラインタブと同一
  - フィルター・ソート: タイムラインタブと同一のUIを適用（共有モードは種別ピルのみ、管理モードはフィルターアイコン+ステータスピル）
  - パートナーの記録をタップすると詳細画面へ遷移（閲覧のみ、編集不可）
  - 管理モード時、承認待ちの記録をタップすると承認操作画面へ遷移（承認 / 拒否。拒否時は理由をコメント）

### 設定タブ
- プロフィール（名前、メールアドレス）
- パートナー管理（未連携時: メールアドレスで招待送信 / 受信した招待の承認、連携済み: パートナー情報表示）
- 管理モード（オン/オフ切り替え。招待者のみ表示。自分・パートナー両方のモードを変更可能。ソロモード時は非表示）
- ログアウト
