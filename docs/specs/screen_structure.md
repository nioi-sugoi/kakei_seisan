## 画面構成

### 認証画面
- メールOTP（メールアドレス入力 → 認証コード送信 → コード入力 → 認証完了）
- ソーシャルログイン（Google / Apple）

### メイン画面

ナビゲーションはボトムタブ。モードによってタブの表示が変わる。各ユーザーが異なるモードで運用可能。

| モード | タイムライン | パートナー | 設定 |
|--------|:-:|:-:|:-:|
| ソロ | ○ | 非表示 | ○ |
| 共有 | ○ | ○（閲覧のみ） | ○ |
| 管理 | ○（ステータス表示） | ○（承認操作あり） | ○ |

### タイムラインタブ
- 上部に残高サマリーを表示（立替合計 − 預り合計 − 精算済み合計）。プラスなら「家計から受け取り」、マイナスなら「家計へ入金」
- 残高サマリー内に「精算する」ボタン
- 記録と精算を同一タイムラインで時系列表示。月ごとのセクション区切り
- 無限スクロール。初回は最新50件を読み込み、スクロールで追加読み込み（50件ずつ）
- 修正・取り消しレコードは作成日時の位置に独立エントリとして表示。「修正」「取消」バッジで視覚的に区別し、元の記録への参照リンクを表示する。元の記録側にも「修正済み」「取消済み」のステータスを表示する
- フィルタ: すべて / 立替 / 預り / 精算
- FAB（＋ボタン）から記録登録フォームへ遷移
- 記録タップで詳細画面へ遷移。詳細画面から修正・取り消しが可能
- 管理モード時は各記録に承認ステータス（承認待ち / 承認済み / 差し戻し）を表示。差し戻された記録にはパートナーのコメントを表示

### 記録登録フォーム
- 種別: 立替 / 預り（セグメント切り替え）
- 金額（日本円）
- 日付（デフォルト: 当日）
- ラベル（自由入力）
- メモ（任意）
- レシート画像（撮影 / ギャラリーから選択、任意、最大2枚）

### 精算実行画面（「精算する」ボタンから遷移）
- 現在の残高を表示
- 精算額を入力（デフォルト: 残高全額、変更可能で部分精算に対応。精算額 > 残高は不可）
- 残高がマイナス（預り超過）の場合は「家計に入金する額」として表現する
- 証跡画像の添付が可能（撮影 / ギャラリーから選択、任意、最大2枚）

### パートナータブ
- パートナーの残高サマリーを表示
- パートナーの記録・精算をタイムラインで閲覧
- フィルタ: すべて / 立替 / 預り / 精算（管理モード時はさらに 承認待ち / 差し戻し で絞り込み可能）
- パートナーの記録をタップすると詳細画面へ遷移（閲覧のみ、編集不可）
- 管理モード時、承認待ちの記録をタップすると承認操作画面へ遷移（承認 / 拒否。拒否時は理由をコメント）

### 設定タブ
- プロフィール（名前、メールアドレス）
- パートナー管理（未連携時: メールアドレスで招待送信 / 受信した招待の承認、連携済み: パートナー情報表示）
- 管理モード（オン/オフ切り替え。招待者のみ表示。自分・パートナー両方のモードを変更可能。ソロモード時は非表示）
- ログアウト
