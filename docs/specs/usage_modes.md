## 利用モード

- **ソロ**: パートナー未連携。自分の記録・精算のみを管理する
- **共有**: パートナー連携済み、管理モードなし。互いの記録・精算を閲覧できる
- **管理**: パートナー連携済み、管理モードあり。記録・精算にパートナーの承認が必要になる

### モードの設定ルール
- 各ユーザーが異なるモードで運用可能（例: User Aは管理モード、User Bは共有モード）
- モードの変更権限はパートナー招待を送信した側（招待者）のみが持つ
- 招待者は自分・パートナー両方のモードを変更できる
- 被招待者はモードを変更できない

### モード切り替え時の挙動
- **共有→管理（オフ→オン）**: 切り替え以降の新規レコードのみ承認が必要。既存レコードには影響なし
- **管理→共有（オン→オフ）**: その時点の承認待ちレコードはすべて自動承認扱いとなり、残高に反映される
